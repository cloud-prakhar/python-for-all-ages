## What is this course?

Welcome to the **Python Basics Course**! Python is one of the most popular programming languages in the world. It's easy to read, fun to write, and used everywhere — from YouTube to NASA, from data science to web development.

Think of Python like a **recipe book for computers** — you write step-by-step instructions and the computer follows them exactly.

Python was created by **Guido van Rossum** in 1991 and has since become the go-to language for beginners and professionals alike.

## How to Run Examples

```bash
# Windows
python example_01_hello_world.py

# Linux / macOS
python3 example_01_hello_world.py
```

Each chapter folder contains:
- `README.md` — Concepts, theory, and key terms
- `example_XX_name.py` — Runnable, commented examples
